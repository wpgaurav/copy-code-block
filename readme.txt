== Description ==
Extends the WordPress Code block to add a "Copy Code" button. When the button is clicked, the code is copied to the clipboard and a success message is displayed.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/copy-code` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
